View time series 2D/3D

 
A single time series is selected and the 2D and 3D scatter diagrams are plotted, where a delay time can also be specified. The delay regards the delay embedding of time series used in the nonlinear analysis of time series, where the 2D and 3D scatter diagrams can be viewed as projections of the underlying attractor.
 

Select one time series
A single time series name can be selected from this list.
 

Draw points / lines
The drawing type to be used in the plots, one of the following: points, lines, both.
 

Delay time
The delay time for the scatter diagrams. The value can be selected by moving the bar horizontally. If a delay t is chosen the 2D scatter diagram is drawn for point pairs (x(i-t), x(i)) and the 3D scatter diagram is drawn for triples (x(i-2t), x(i-t), x(i)).
 

2D scatter diagram
The plot of point pairs (x(i-t), x(i)) of the selected file, where t is the selected delay, is shown in a figure with the selected line or/and point type (see figure below). 

 

 

3D scatter diagram
The plot of point triples (x(i-2t), x(i-t), x(i)) of the selected file, where t is the selected delay, is shown in a figure with the selected line or/and point type (see figure below).
