Table of Measures vs Time Series

 
A table of selected measures and time series is shown in a separate data window and can be further saved in a file.

One or more time series can be selected from the time series list having the title Select one or more time series.  In the same way, one or more measures can be selected from the measure list having the title Select one or more measures. For measures defined in terms of one or more measure specific parameters, the measure name regards a specified set of parameters (one parameter value for each name). A subset of time series names or measure names can be selected from the respective lists with standard use of the mouse: a click on a name and Shift^click on a another name further down in the list selects all names between the former and the latter; a click on a name and subsequent Ctrl^clicks on other names select these names including the first name; Ctrl^a selects the whole list.

 

View selected table
Provided that a list of time series names and measures names, or even a single time series name or/and measure name, are selected, by pressing this button a separate window of the table will be displayed (as the one shown below). The table contains the measure names in the first column and the time series names in the first row, and a value at a cell corresponds to the measure of this row and the time series of this column. Note that it is not possible to edit this table, but it is possible to change the width of the columns to display best the content.

 

Save selected table
This button is active (the string of the button is emphasized) only when the "View selected table" operation has been successfully run. When pressing this button a matlab dialogue window appears (as shown below) and the user can specify the file to save the table of the selected measure values (rows) for the selected time series (columns). The file can be plain ascii, excel (.xls) and matlab data file (.mat).

 

Running Messages
A text box displaying messages for the running operations.
 

Exit
By pressing this button the window of "Table of Measures vs Time Series" will disappear and the user will be moved to the "View Measures" window. Note that any created tables here will remain until the user closes the table windows.
