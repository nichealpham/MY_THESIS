Plot of measure vs parameter
This operation plots in a figure a measure as a function of a varying parameter (at least two values of the specified parameter should exist) for selected time series.  This is the classical visualization for the dependence of a measure on its parameter, allowing for superimposing for different time series. This operation can be applied only to measures bearing at least one varying parameters.

One or more time series can be selected from the time series list having the title Select two or more time series.

Before selecting from the measure list the Parameter must be chosen among the listed parameters (t, m, h, f, k, q, b, s, r, a, w, see "Measure Parameter Name Notation" in the main menu). Upon the selection of the parameter, the list of measure names is restricted to the measure names that include this parameter (measures that are dependent on this parameter). If for example, the "delay (t)" is selected, the label on the top of the measure list reads Select measures of parameter type "delay (t)" and the list contains all measure names that contain this parameter (the measure names contain the character 't' followed by a number). If no such a measure name exists, the list is empty and this operation cannot run for this parameter.

A subset of time series names and measure names can be selected from the respective lists with standard use of the mouse: a click on a name and Shift^click on a another name further down in the list selects all names between the former and the latter; a click on a name and subsequent Ctrl^clicks on other names select these names including the first name; Ctrl^a selects the whole list.

Tips: For the selection of a subset of measure names from the list, click on the measure of this subset appearing first in the list and then Shift^click at the last measure in the list. In this way, all measures from the first selected to the last measure in the list will be marked and from those only the ones that have the same measure name as the first and differ only to the value of the specified parameter will be processed.  

 

Parameter
The user can choose one of the following parameters (for details see "Measure Parameter Name Notation" in the main menu)

delay (t)
embedding dimension (m)
prediction time (h)
fraction for test set (f)
number of neighbors (k)
truncation parameter (q)
number of bins (b)
upper/low scaling ratio (s)
radius (r)
file order (a)
offset for window length (w)

If there are measures in the list of measure names that are defined in terms of the selected parameter, they are displayed in the list of measures.
 

View selected plot
Provided that a list of at least two measure names of the same measure and parameter (differing only in the parameter value) and one or more time series names are selected, by pressing this button a separate figure window will be displayed (as the one shown below). The graph of measure vs  parameter is displayed with a distinct point or line type for each time series. When up to 5 time series names are selected, they are listed in a legend together with their corresponding drawing type, otherwise they are listed in the "Running messages" text box.
 

draw lines / points
The user can choose one of the following: Lines, Points, Both. This is the drawing type for the measure vs parameter plot and a different drawing type is used for each time series, as explained in the "Running messages" text box.
 

Running Messages
A text box displaying messages for the running operations.
 

Exit
By pressing this button the window of "Plot of measure vs parameter" will disappear and the user will be moved to the "View Measures" window. Note that any created figures will remain until the user exits from the "View Measures" window.
